A specific guideline which states that 20% of the components of a situation account for 80% of the outcomes. Return to text.
activity
Any part of a set of actions that generates work to accomplish a job, problem, or assignment. Return to text.

A report that documents the expected tasks and estimated time for completing a given activity, based on observation of or interviews with an individual performing the activity. Return to text.

An infrequent decision that can be made quickly at an individual level because the risks involved is relatively small. See Module 3, Chapter 7.
agile management
An approach to business effectiveness focused on finding and seizing opportunities to improve operations and processes. Return to text.
AI
— artificial intelligence. See Module 1, Chapter 1.

The breaking down of a problem into separate manageable parts, and finding the best workable solution. See Module 3, Chapter 10.

A forecasting model that uses the arithmetic average of a series of values as the forecast value for the next future period. See Module 3, Chapter 8.

The area of computer science that emphasizes the development of intelligent machines that work, learn, and react like humans. See Module 1, Chapter 1.

A consistent error in research outcomes that arises when people perform better because they know they are being studied. See Module 3, Chapter 8.

A performance monitoring tool that displays a set of KPIs and compares the value to performance standards. Return to text.
